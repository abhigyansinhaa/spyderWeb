A dark-themed Next.js portfolio showcasing data engineering and ML projects, experience, skills, and certifications.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

- **Vercel**: Connect this repo to [Vercel](https://vercel.com) for zero-config deployment.
- **Static export**: Add `output: 'export'` in `next.config.ts`, then run `npm run build` and deploy the `out` folder (e.g. GitHub Pages).

## Customize

- Replace the "Photo" placeholder in `components/About.tsx` with your profile image.
- Update project GitHub links in `components/Projects.tsx` to point to your actual repos.
